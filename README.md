# My Alfred Workflows

## 1. iTerm Workflow

### 1.1. Pre-requisites

- iTerm application

### 1.2. Usage

To open the currently selected directory in the Finder app, type on Alfred:

```shell
iterm
```

To choose and open a directory, type on Alfred:

```shell
iterm* <YOUR DIRECTORY>
```

## 2. Jupyter Lab Workflow

Works very similarly to the iTerm Workflow, except opens the directory using [Jupyter Lab](https://jupyterlab.readthedocs.io/en/stable/).

### 2.1. Pre-requisites

- iTerm application
- `jupyter lab` as an executable command from iTerm

### 2.2. Usage

To open the currently selected directory in the Finder app, type on Alfred:

```shell
jupyter
```

To choose and open a directory, type on Alfred:

```shell
jupyter* <YOUR DIRECTORY>
```
